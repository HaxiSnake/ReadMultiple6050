<h1>ReadMultiple6050</h1>
<p>使用串口同时读取多个6050</p>
<h2>硬件要求：</h2>
<p>mpu-6050</p>
<p>蓝牙串口透传</p>
<h2>软件要求：</h2>
<p>System:Win 10</p>
<p>Python 3.0</p>
<p>Python package:numpy pyserial tkinter</p>
<P>------------------------------------</p>
<P>运行main.py即可,选择串口号之后点击Open打开串口,点击Record记录数据,数据保存在Data文件夹下。</p>
